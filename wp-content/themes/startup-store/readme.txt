== Description ==

Startup Store is an elegant and trendy eCommerce theme. It's built using the bootstrap for the frontend interface and ensures the beautiful, fast, lightweight, responsive WordPress theme. Startup Store will recommend using e-commerce, beauty products, women's fashion, smart home products, sports, toy shops, gadgets stores, jewelry shops, men's clothing, electronics, furniture, retail, digital products, and preferably ideal for any eCommerce sites. Toclearfiy, Startup Store is a child theme of Startup Shop,  Startup Store Compatible with Gutenberg, and the most popular page builder Elementor. You can visually build a website with drag and drop blocks and widgets. Startup-Store comes with demo content import plugins to get a single click many Header & footer layouts, predesigned Home & Inner Pages template. Startup  Store is tested with all top-rated plugins like WooCommerce, Yoast, Contact form 7, Dokan, WordPress's Jetpack, bbPress, etc. Are you looking for a Multipurpose Shop theme? Look no further! Browse the demo to see that it's the only theme you will ever need: https://athemeart.dev/wp/startup/

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
